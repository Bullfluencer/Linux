Linux

CLI. 1 om index.html te vinden. Als eerste open je de terminal. Dan type je cd /var/www/html, vervolgens type je ls en dan zie je index.html. 2 om alle files met .sh te vinden. Heb je een simpele command namelijk whereis .sh, deze command laat je zien waar alle documenten met .sh te vinden zijn. 3 om echo te vinden. Gebruik je dezelfde command namelijk whereis echo, deze command laat je zien waar echo gelocate is. GUI 1 Je kan in linux makkelijk door je bestanden zoeken in de verkenner, daar zie je alle directory’s en subdirectory’s. Verder kan je ook zoeken door op het icoontje met 9 stippen te klikken Dan krijg je alle programma’s te zien en een zoekbalk. 2 om een search engine te instaleren met GUI dan open je eerst Ubuntu software center. Zoek dan de search engine naar keus, en installeer het.
