Cli scripts Windows.

Dit geld voor de mensen op windows. 1. Maak een folder in je cloud genaamd cliScripts. 2. Daarin vervolgens een Windows folder. 3. Kopieer je .txt van de linux en maak weer een cliScripts text bestand in de cliScripts folder. 4. Download het .bat bestand en zet die in de Windows folder. Open cmd en navigeer naar de 01makefiles.bat door cd commando. Type dan 01makeFiles 5. Hernoem het batch bestand naar 02makeFolders. Haal alles na de for ( weg behalve de 2 set commands. Vervolgens doe je daar mkdir !lastname!!firstname! En daaronder een echo met wat we doen. Type dan weer in cmd 02makeFolders. 6. Sla 02makeFolders op als 03moveFiles. Haal alles behalve de set commands weg. Type dan move ReadmeFor!firstname!!lastname! !lastname!!firstname!. En een echo met wat we doen. En type tot slot weer 03moveFiles. 7. Sla 03moveFiles op als 04renameFolders. Verander op regel 12 de tokens van 1-2 naar 1-3. Voeg onder de set commands ook set domainname=%%c. Haal de vorige commands weg in de for loop en type ren !!firstname!lastname! !domainname!. Vervolgens type je in cmd 04renameFolders. 8. Hernoem 04renameFolders naar 05deleteFilesAndFolders. Haal weer alles weg behalve de set commands. Type vervolgens de command rmdir !lastname!!firstname! En voer het script uit met 05deleteFilesAndFolders. 9. Noem 05deleteFilesAndFolders naar 06makeWebFilesAndFolders. Verander de tokens token 1-3 naar 1-6. Voeg ook de variabelen toe door set color1=%%D te typen enzv. Type in de for loop de commands mkdir (make directory) voor alle bestanden die je wilt toevoegen. Voer vervolgens het script uit met 06makeWebFilesAndFolders in cmd.
