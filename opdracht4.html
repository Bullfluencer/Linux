Introductie CLI (command line interface).

command+shift+T opent de Command Line Interface. De command man man opent een gids die je kan volgen die je helpt met allerlei commands. Man ls geeft informatie over de command ls, die sorteert informatie over de files in de directory. Ls -S, laat de files in je dir zien. Ls -lh laat de grootte van de files zien per folder. Ls -R sorteert de subdirectories achteruit. Ls -LS laat ook files in je directory zien. Tab laat je zien welke commands je kan invoeren met de letters die je al hebt getypt, en verder kan je er ook commands mee autocompleten. Je krijgt de / te zien als je pwd invoert na cd / verder zie je home/user als je cd ~ en dan pwd. Command history laat je alle commands zien die je hebt ingevoerd.
