Linux.

Linux naar html klik je eerst op de verkenner Dan ga je naar het mapje “var” Vervolgens klik je op www (die staat er pas nadat je apache hebt gedownload). en dan naar html. Als er geen www bestand is dan moet je naar de terminal en sudo apt-get update, vervolgens moet je dan sudo apt-get install apache2 invoeren en op y drukken. om een bestand plaatsen: maak je eerst eentje aan in je IDE bijvoorbeeld VSC. Dat bestand sla je dan op in de html folder. Je moet eerst sudo chmod 777/var/www in voeren om toestemming te krijgen om iets op te slaan. 1. Openen 2. Prompt Cd -> var/www/html 3. Touch helloworld.txt maken en openen met nano.
